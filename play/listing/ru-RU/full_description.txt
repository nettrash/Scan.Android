Scan читает и создаёт распространённые 1D- и 2D-штрихкоды — и понимает, что внутри, а не просто отдаёт сырой текст.

Наведите камеру на Wi-Fi QR — увидите имя сети, тип защиты и кнопку «Открыть настройки Wi-Fi». Отсканируйте платёжку SEPA — IBAN, получатель и сумма появятся отдельными строками с копированием в одно касание. Отсканируйте визитку — система сразу откроет форму «Новый контакт». Отсканируйте QR события — откроется системный экран «Добавить событие» с заголовком, местом и временем в вашем часовом поясе.

ПРИВАТНОСТЬ ПО УМОЛЧАНИЮ

Scan не собирает, не передаёт и не делится никакими данными. Модель распознавания работает целиком на устройстве — кадры с камеры обрабатываются в памяти и никогда не сохраняются и не уходят в сеть. История хранится в приватной базе приложения. Никаких аккаунтов, никакой телеметрии, никаких рекламных SDK, никаких трекеров.

ЖИВОЕ СКАНИРОВАНИЕ

• EAN-8, EAN-13, UPC-A, UPC-E
• Code 39, Code 93, Code 128, ITF, Codabar

• Фонарик и угловые скобки рамки, которые «прилипают» к найденному коду

УМНЫЙ РЕЗУЛЬТАТ

Распознаваемые форматы — каждое поле копируется одним касанием:

• Веб — URL, mailto, tel, sms / smsto
• Подключение — Wi-Fi (SSID, пароль, тип защиты, скрытая сеть)
• Геолокация — geo:, открывается в Картах
• Контакты — vCard 3.0, MECARD
• Календарь — iCalendar VEVENT (UTC, часовой пояс, события на весь день)
• Аутентификация — otpauth:// для приложений-аутентификаторов
• Товарные коды — EAN / UPC / ITF
• Криптовалюты — Bitcoin (BIP-21), Ethereum (EIP-681 с chain ID), Litecoin, Bitcoin Cash, Dogecoin, Monero, Cardano, Solana, Lightning (BOLT-11)
• Банковские платежи — EPC SEPA / GiroCode (ЕС), Swiss QR-bill, Czech SPD, Slovak Pay by Square, российская универсальная платёжка (ST00012 / ST00011), EMVCo Merchant QR с разбором вложенных шаблонов для Pix, PayNow, PromptPay, CoDi, UPI-через-EMVCo, DuitNow, QRIS, FPS, NAPAS, NETS, индийский UPI, Bezahlcode, сербский NBS IPS QR (Prenesi)
• Мобильные платежи — Swish, Vipps, MobilePay, Bizum, iDEAL
• Чеки — российский чек ФНС, сербский фискальный чек SUF

УМНЫЕ ДЕЙСТВИЯ

• URL — открыть в браузере
• Email / телефон / SMS — открыть нужное системное приложение
• Wi-Fi — скопировать пароль, открыть настройки Wi-Fi
• Координаты — открыть в Картах
• vCard — добавить в Контакты через системный редактор
• iCalendar — добавить в Календарь через системный редактор
• Крипто / UPI / региональные платежи — открыть в установленном кошельке или банковском приложении
• Сербский фискальный чек — проверить на официальном сайте PURS
• Любое поле — скопировать; полный код — поделиться через системное «Поделиться»
